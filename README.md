### Голосовой помощник. Инструкция ###

##### Зависимости: ####
###### python 3.6.5 ###### 

```
скачать https://www.python.org/downloads/release/python-365/
```

###### менеджер пакетов PIP ###### 

```
в косноли выполнить python -m pip install -U pip
```

###### зависимости проекта ######

```
в консоли выполнить python -m pip install -r requirements.txt
```

##### Использование #####

```
в консоле запустить voice_helper.py
сказать что-нибудь в микрофон
```
